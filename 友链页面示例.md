## 友链说明

欢迎互换友链！在我们有一定了解之后可以交换友链。

### 交换要求

你的网站需要满足以下条件：
- 合法的、非营利性、无商业广告
- 有实质性原创内容的 HTTPS 站点
- 网站能够正常访问

### 如何申请友链？

请前往 [友链管理仓库](https://github.com/你的用户名/friends/issues) 提交 Issue，按照模板填写你的网站信息即可。

### 本站信息

如果你想添加本站友链，可以使用以下信息：

```json
{
  "title": "晓天的浮空岛",
  "url": "https://你的域名.com/",
  "description": "晓天的个人博客",
  "icon": "https://res.cloudinary.com/dekmp7ltk/image/upload/v1735656517/%E5%B0%8F%E5%A4%B4%E5%B8%A6%E8%88%8C%E5%A4%B4png_jc9hbt.png",
  "snapshot": "",
  "feed": "https://你的域名.com/atom.xml"
}
```
